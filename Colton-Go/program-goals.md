# command line budget/balance tracker
basic crud operations for expenses. 
Record expense amount, description, date
sum all expenses, sum expenses by date range or by month
import expenses from a csv file


## basic functionalities
- create an expense 
    - includes a description, price, and date

- view all current expenses
    - display all fields
    - possibly customize to show only specific fields (using cli flags?)

- update an expense
    - edit a description, price, or date

- delete an expense
    - remove a wrong expense

- get overall data from your 'expenses' 
    - how much have you spent?
    - how much have you spent in a specific time frame?

- stretch
    - import expenses from a csv file
    - sum expenses by month or custom date range
    - add incomes to enable money in/money out tracking. 
    - budgeting?
